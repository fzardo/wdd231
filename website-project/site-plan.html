Song Stash

Project Description

The user will be able to check all music featured on the website, filter information, and check details about the author or band.

Reason

I particularly like music and want to experiment with making an alternative online library.

Scenarios

  • How can I filter music by genre or decade?
  • Where can I find details about a specific artist or band?

Color Schema

  • Yellow - Used for headings and primary accents.
  • Red - Used for highlights and call-to-action buttons.
  • #9dd3ff - Background color.
  • Black - Used for body text.

Typography

  • Heading Font: Verdana, sans-serif (used for all headings).
  • Body Font: Verdana, sans-serif (used for paragraph text and descriptions).
